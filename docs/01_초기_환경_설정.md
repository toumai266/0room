# 초기 환경 설정 및 세팅

## 프로젝트 구조
```

├── app.py
├── templates/
│   └── index.html
├── docs/
```

## 설정 내용
- Flask 프로젝트 초기화
- 가상환경(.venv) 생성 및 활성화
- Flask 라이브러리 설치
- 기본 라우팅(/) 설정
- templates 폴더 생성 및 index.html 파일 생성
- Flask 개발 서버 실행 환경 구축 (flask run)
- 문서화 폴더(`docs/`) 구조화
  - PRD.md: 프로젝트 요구사항 정의서
  - 개발일지.md: 일일 개발 로그
